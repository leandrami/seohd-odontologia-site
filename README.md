# 📌 Sobre o projeto

Este projeto foi desenvolvido como um **site institucional para a clínica SEOHD Odontologia**, com o objetivo de apresentar os serviços oferecidos e facilitar o acesso às informações da empresa.

O design foi pensado para transmitir **cuidado, confiança e profissionalismo**, valores essenciais na área da saúde.

O layout busca oferecer:

- navegação simples
- leitura clara das informações
- boa organização de conteúdo
- identidade visual alinhada à área odontológica

---

# 🎨 Design e UX

Durante o desenvolvimento foram aplicados princípios de:

- Hierarquia visual
- Clareza de informação
- Legibilidade
- Organização de conteúdo
- Experiência do usuário

O projeto também buscou manter **consistência visual e acessibilidade básica**, facilitando a navegação para diferentes perfis de usuários.
